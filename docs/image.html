API Docs for project image2html

IMAGE2VIDEO. API for image

You can make request with any browser, wget or curl. 
curl -i -X POST -F in='{ "apikey":"315820", "action":"list", "project_id": "150618673841be33720c67f96493b0757f34357fa759b6ce3e"}' http://your_ip/image2video/api/image.php
curl -i -X POST  -H "Content-Type: multipart/form-data" -F 'in=@/tmp/image.json' http://your_ip/image2video/api/image.php
API answer - json string with field status and additional values, for example: 
 
{
	"status": "ok", 
	"count":"12"
}
or 
 
{
	"status": "error",
	"errorno": "4000", 
	"error": "Don't recived input parameters" 
	}
There are possible next actions: list, add, remove, move .
