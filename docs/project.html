API Docs for project image2html

IMAGE2VIDEO. API for projects

All request will be doing with web interface, parameter 'id' with json string, someting like: 
http://your_ip/image2video/api/project.php?in={%22action%22:%22set_status%22,%22id%22:%22150583385512136%22,%22status%22:%22paused%22}
You can make such request with any browser, wget or curl. API answer - json string with field status and additional values, for example: 
 {"status": "ok", "id":"1234567"}
or 
 {"status": "error","errorno": "4000", "error": "Don't recived input parameters" }
There are possible next actions: list, count, clone, remove .

list

By default, return list of all projects.
Required parameters:
  • "apikey":"315820"
  • "action":"list"
Optional parameters:
  • "id":"1234567" - show info only for project with this id
Samples: Request: 
{	
	"apikey":"315820",
	"action":"list",
	"id":"150583385512136"
}
Answer: 
{
    "status": "ok",
    "rows": [
        {
            "project_name": "New project September 18, 2017, 6:39 pm",
            "project_id": "1505760001d5ea1ebf56c2ee9a7c0973a57a2efaff28b3819e"
        },
        {
            "project_name": "Clone of New project September 7, 2017, 5:14 pm",
            "project_id": "1505399648dc78d45b38c7490cd65d6933129ed37fa903a00e"
        }
    ]
}

count

Return the count of projects
Required parameters:
  • "apikey":"315820"
  • "action":"count"
Optional parameters:
Request: 
{
	"apikey":"315820",
	"action":"count"
}
Answer: 
{
    "status": "ok",
    "count": 5
}

add

Required parameters:
  • "apikey":"315820"
  • "action":"add"
  • "name":"New project"
Optional parameters:
Samples: Request: 
{
	"apikey":"315820",
	"action":"add",
	"name":"New project 1"
}
Answer: 
{
    "status": "ok",
    "id": "150618658947796a88478d28f0ade0534bdb7ac02ebc876961"
}

{
