API Docs for project image2html

IMAGE2VIDEO. API for projects

You can make request with any browser, wget or curl. 
curl -i -X POST -F in='{ "apikey":"315820", "action":"list", "project_id": "150618673841be33720c67f96493b0757f34357fa759b6ce3e"}' http://your_ip/image2video/api/project.php
curl -i -X POST  -H "Content-Type: multipart/form-data" -F 'in=@/tmp/project.json' http://your_ip/image2video/api/logo.php
API answer - json string with field status and additional values, for example: 
 
{
	"status": "ok", 
	"count":"12"
}
or 
 
{
	"status": "error",
	"errorno": "4000", 
	"error": "Don't recived input parameters" 
	}
There are possible next actions: list, count, clone, remove .

list

By default, return list of all projects.
Required parameters:
  • "apikey":"315820"
  • "action":"list"
Optional parameters:
  • "project_id":"1234567" - show info only for project with this id
Samples: Request: 
{	
	"apikey":"315820",
	"action":"list",
	"project_id":"150583385512136"
}
Answer: 
{
    "status": "ok",
    "rows": [
        {
            "project_name": "New project September 18, 2017, 6:39 pm",
            "project_id": "1505760001d5ea1ebf56c2ee9a7c0973a57a2efaff28b3819e"
        },
        {
            "project_name": "Clone of New project September 7, 2017, 5:14 pm",
            "project_id": "1505399648dc78d45b38c7490cd65d6933129ed37fa903a00e"
        }
    ]
}

count

Return the count of projects
Required parameters:
  • "apikey":"315820"
  • "action":"count"
Optional parameters:
Request: 
{
	"apikey":"315820",
	"action":"count"
}
Answer: 
{
    "status": "ok",
    "count": 5
}

add

Add new project
Required parameters:
  • "apikey":"315820"
  • "action":"add"
  • "name":"New project"
Optional parameters:
Samples: Request: 
{
	"apikey":"315820",
	"action":"add",
	"name":"New project 1"
}
Answer: 
{
    "status": "ok",
    "project_id": "150618658947796a88478d28f0ade0534bdb7ac02ebc876961"
}

clone

Clone existing project
Required parameters:
  • "apikey":"315820"
  • "action":"clone"
  • "project_id":"123456576"
Optional parameters:
  • "name":"Name for cloned project"
Samples: Request: 
{
	"apikey":"315820",
	"action":"clone",
	"name":"New clone of project 1"
}
Answer with id of cloned project: 
{
    "status": "ok",
    "project_id": "150618673841be33720c67f96493b0757f34357fa759b6ce3e"
}

remove

Remove project
Required parameters:
  • "apikey":"315820"
  • "project_id":"1234567"
Optional parameters:
Samples: Request: 
{	
	"apikey":"315820",
	"action":"remove",
	"project_id":"150583385512136"
}
Answer: 
{
    "status": "ok"
}
